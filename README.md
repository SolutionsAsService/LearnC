## Folder Structure

- `basic_concepts/` - Hello World, Variables, Conditionals, Loops
- `functions/` - Functions and their usage
- `arrays/` - Understanding arrays
- `pointers/` - Introduction to pointers
- `structs/` - Using structs for data grouping
